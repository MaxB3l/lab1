#  Обработка изображений на Flask

Веб-приложение для рисования креста на изображениях с визуализацией распределения цветов.

## Демо

https://MaxBelyakov.pythonanywhere.com

## Возможности

- Загрузка изображений (PNG, JPG)
- Рисование креста (вертикальный/горизонтальный)
- Выбор цвета креста
- Статистика цветов (яркость, проценты RGB)
- Гистограммы распределения цветов
- Защита от ботов (Google reCAPTCHA v2)

##  Технологии

- **Обработка изображений:** Pillow
- **Формы:** Flask-WTF
- **Дизайн:** Bootstrap 5
- **Хостинг:** PythonAnywhere
- **CI/CD:** GitHub Actions
